# 🇬🇧 🇺🇸 Ra
A game where you must use sunlight and mirrors to light up the Pharaoh's Sarcophagus.

The sun position is calculated by the same algorythm used in the [Sun Motion Simulator](https://github.com/7331Squall/caminhos-do-sol-v0).

# 🇵🇹 🇧🇷 Ra
Um jogo onde você deve usar o sol e espelhos para iluminar o sarcófago do faraó.

A posição do sol é calculada com o mesmo algoritmo usado pelo [Caminhos do Sol](https://github.com/7331Squall/caminhos-do-sol-v0).
